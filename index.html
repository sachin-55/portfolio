Unit Calculator
js|react|redux|node|express
Unit Calculator is an utility website. Here user can calculate units and cost of a specific electicity sub-meters. User is required to provide two meter readings (previous and current readings) and unit rate. User can create his/her account to save monthly readings of specified sub-meters so that user can just select months duration to calculate units and cost.
